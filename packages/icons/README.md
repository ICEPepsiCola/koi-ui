# @koi-ui/icons

Vendored Heroicons React components for Koi UI.

```tsx
import { XCircleIcon } from '@koi-ui/icons/24/outline';
```

The icon source SVGs come from the [`heroicons`](https://www.npmjs.com/package/heroicons) package (v2.2.0) and are transformed into local TSX components via `pnpm icons:generate`. See `LICENSE.heroicons` for upstream attribution.
