
# Box

## Basic usage

```tsx preview
import { Box, Text } from '@koi-ui/core';
export default () => (
  <Box p={6} border rounded="md">
    <Text>Container</Text>
  </Box>
);
```

## Padding

```tsx preview
import { Box, Text } from '@koi-ui/core';
export default () => (
  <div className="space-y-3">
    <Box p={2} border><Text size="sm">p=2</Text></Box>
    <Box p={3} border><Text size="sm">p=3</Text></Box>
    <Box p={4} border><Text size="sm">p=4</Text></Box>
    <Box p={10} border><Text size="sm">p=10</Text></Box>
  </div>
);
```

## Radius & background

```tsx preview
import { Box, Text, Space } from '@koi-ui/core';
export default () => (
  <Space wrap size="middle">
    <Box p={4} rounded="none" border><Text size="sm">none</Text></Box>
    <Box p={4} rounded="sm" border><Text size="sm">sm</Text></Box>
    <Box p={4} rounded="md" border><Text size="sm">md</Text></Box>
    <Box p={4} rounded="lg" border><Text size="sm">lg</Text></Box>
    <Box p={4} bg="muted"><Text size="sm">muted</Text></Box>
    <Box p={4} bg="primary"><Text size="sm" className="text-primary-foreground">primary</Text></Box>
  </Space>
);
```
