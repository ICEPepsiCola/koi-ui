# Popconfirm

## Basic usage

```tsx preview
import { Popconfirm, Button } from '@koi-ui/core';
export default () => <Popconfirm title="Delete this item?" onConfirm={() => {}}><Button variant="destructive">Delete</Button></Popconfirm>;
```

## API

### Popconfirm

| title | Title | `ReactNode` **(required)** | `-` |
| cancelText | - | `string \|тАЛ undefined` | `Cancel` |
| confirmText | - | `string \|тАЛ undefined` | `OK` |
| description | Description | `ReactNode` | `-` |
| disabled | Disabled | `boolean \|тАЛ undefined` | `-` |
| okType | - | `enum` | `primary` |
