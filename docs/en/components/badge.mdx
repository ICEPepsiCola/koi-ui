# Badge

## Basic usage

```tsx preview
import { Badge, Button } from '@koi-ui/core';
export default () => <Badge count={5}><Button>Message</Button></Badge>;
```

## API

### Badge

| Property | Description | Type | Default |
| :---: | :---: | :---: | :---: |
| count | Count | `number \|​ undefined` | `-` |
| dot | - | `boolean \|​ undefined` | `-` |
| max | - | `number \|​ undefined` | `99` |
| showZero | - | `boolean \|​ undefined` | `false` |
| size | Size | `enum` | `-` |
| variant | Variant | `enum` | `-` |
