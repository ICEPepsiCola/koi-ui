# Input

## Basic usage

```tsx preview
import { Input } from '@koi-ui/core';
export default () => <Input placeholder="Enter text" />;
```

## Error state

```tsx preview
import { Input, Stack } from '@koi-ui/core';
export default () => (
  <Stack gap={3}>
    <Input placeholder="Normal" />
    <Input placeholder="Invalid" error="Enter valid content" />
    <Input placeholder="Disabled" disabled />
  </Stack>
);
```

## Controlled input

```tsx preview
import { Input } from '@koi-ui/core';
import { useState } from 'react';
export default () => {
  const [value, setValue] = useState('');
  return <Input value={value} onChange={setValue} placeholder="Synced input" />;
};
```

## clearable

```tsx preview
import { Input } from '@koi-ui/core';
import { useState } from 'react';
export default () => {
  const [value, setValue] = useState('可清除Content');
  return <Input value={value} onChange={setValue} clearable placeholder="可清除输入" />;
};
```

## API

### Input

| Property | Description | Type | Default |
| :---: | :---: | :---: | :---: |
| clearable | - | `boolean \|​ undefined` | `false` |
| error | Error message | `string \|​ boolean \|​ undefined` | `-` |
| onChange | Change callback | `((value: string) => void) \|​ undefined` | `-` |
| size | Size | `enum` | `-` |
