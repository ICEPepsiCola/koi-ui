# Pagination

## Basic usage

```tsx preview
import { Pagination } from '@koi-ui/core';
import { useState } from 'react';
export default () => { const [p,setP]=useState(1); return <Pagination current={p} total={50} onChange={setP} />; };
```

## API

### Pagination

| Property | Description | Type | Default |
| :---: | :---: | :---: | :---: |
| current | Current | `number` **(required)** | `-` |
| total | Total | `number` **(required)** | `-` |
| className | Class name | `string \|‚Äã undefined` | `-` |
| onChange | Change callback | `((page: number) => void) \|‚Äã undefined` | `-` |
