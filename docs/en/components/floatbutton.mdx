# FloatButton

## Basic usage

```tsx preview
import { FloatButton, Icon } from '@koi-ui/core';
export default () => <FloatButton icon={<Icon name="plus" />} />;
```

## API

### FloatButton

| Property | Description | Type | Default |
| :---: | :---: | :---: | :---: |
| description | Description | `ReactNode` | `-` |
| icon | Icon | `ReactNode` | `-` |
| placement | Placement | `enum` | `-` |
| size | Size | `enum` | `-` |
| variant | Variant | `enum` | `-` |
