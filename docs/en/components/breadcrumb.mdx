# Breadcrumb

## Basic usage

```tsx preview
import { Breadcrumb } from '@koi-ui/core';
export default () => <Breadcrumb items={[{ title: 'Home' }, { title: 'List' }, { title: 'Details' }]} />;
```

## API

### Breadcrumb

| Property | Description | Type | Default |
| :---: | :---: | :---: | :---: |
| items | Items | `BreadcrumbItem[]` **(required)** | `-` |
| separator | - | `ReactNode` | `/` |
