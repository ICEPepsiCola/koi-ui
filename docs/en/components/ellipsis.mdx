# Ellipsis

## Basic usage

```tsx preview
import { Ellipsis } from '@koi-ui/core';
export default () => <Ellipsis content="This is a long paragraph used to demonstrate text ellipsis: when content exceeds the container width, the overflow is truncated with an ellipsis so lists and cards stay readable." rows={1} />;
```
