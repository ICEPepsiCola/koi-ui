# Skeleton 骨架屏

## 基础用法

```tsx preview
import { Skeleton } from '@koi-ui/core';
export default () => <Skeleton active paragraph={{ rows: 3 }} />;
```

## API

### Skeleton

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| active | - | `boolean \|​ undefined` | `true` |
| height | 高度 | `string \|​ number \|​ undefined` | `-` |
| variant | 变体 | `enum` | `-` |
| width | 宽度 | `string \|​ number \|​ undefined` | `-` |

### SkeletonGroup

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| avatar | - | `boolean \|​ undefined` | `false` |
| className | 类名 | `string \|​ undefined` | `-` |
| rows | 行数 | `number \|​ undefined` | `3` |
