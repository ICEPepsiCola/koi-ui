# Tooltip 文字提示

## 基础用法

```tsx preview
import { Tooltip, Button } from '@koi-ui/core';
export default () => <Tooltip content="提示文字"><Button>悬停</Button></Tooltip>;
```

## API

### Tooltip

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| content | 内容 | `ReactNode` **(必填)** | `-` |
| className | 类名 | `string \|​ undefined` | `-` |
| defaultOpen | - | `boolean \|​ undefined` | `false` |
| open | 是否显示 | `boolean \|​ undefined` | `-` |
| placement | 弹出位置 | `enum` | `top` |
| variant | 变体 | `enum` | `-` |
