# Collapse 折叠面板

## 基础用法

```tsx preview
import { Collapse } from '@koi-ui/core';
export default () => <Collapse items={[{ key:'1', label:'面板1', children:'内容1' }, { key:'2', label:'面板2', children:'内容2' }]} />;
```

## API

### Collapse

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| items | 数据项 | `CollapseItem[]` **(必填)** | `-` |
| accordion | - | `boolean \|​ undefined` | `false` |
| activeKeys | - | `string[] \|​ undefined` | `-` |
| defaultActiveKeys | - | `string[] \|​ undefined` | `[]` |
| onChange | 变化回调 | `((keys: string[]) => void) \|​ undefined` | `-` |
