# Dropdown 下拉菜单

## 基础用法

```tsx preview
import { Dropdown, Button } from '@koi-ui/core';
export default () => (
  <Dropdown trigger={<Button>下拉菜单</Button>} items={[{ key: '1', label: '选项一' }, { key: '2', label: '选项二' }]} />
);
```

## 禁用项

```tsx preview
import { Dropdown, Button } from '@koi-ui/core';
export default () => (
  <Dropdown
    trigger={<Button>操作菜单</Button>}
    items={[
      { key: 'edit', label: '编辑' },
      { key: 'copy', label: '复制' },
      { key: 'delete', label: '删除', disabled: true },
    ]}
  />
);
```

## API

### Dropdown

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| items | 数据项 | `DropdownItem[]` **(必填)** | `-` |
| trigger | 触发元素 | `ReactNode` **(必填)** | `-` |
| disabled | 是否禁用 | `boolean \|​ undefined` | `-` |
| onSelect | 选择回调 | `((key: string) => void) \|​ undefined` | `-` |
| placement | 弹出位置 | `enum` | `-` |
| responsive | 是否响应式 | `boolean \|​ undefined` | `true` |
| title | 标题 | `ReactNode` | `-` |
