# Segmented 分段控制器

## 基础用法

```tsx preview
import { Segmented } from '@koi-ui/core';
import { useState } from 'react';
export default () => { const [v,setV]=useState('a'); return <Segmented value={v} onChange={setV} options={[{label:'列表',value:'a'},{label:'卡片',value:'b'}]} />; };
```

## API

### Segmented

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| options | 选项列表 | `SegmentedOption[]` **(必填)** | `-` |
| block | - | `boolean \|​ undefined` | `-` |
| className | 类名 | `string \|​ undefined` | `-` |
| defaultValue | 默认值 | `string \|​ undefined` | `-` |
| onChange | 变化回调 | `((value: string) => void) \|​ undefined` | `-` |
| size | 尺寸 | `enum` | `-` |
| value | 当前值 | `string \|​ undefined` | `-` |
