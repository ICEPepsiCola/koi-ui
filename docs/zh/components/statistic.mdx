# Statistic 统计数值

## 基础用法

```tsx preview
import { Statistic } from '@koi-ui/core';
export default () => <Statistic title="活跃用户" value={112893} />;
```

## API

### Statistic

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| precision | - | `number \|​ undefined` | `-` |
| prefix | - | `ReactNode` | `-` |
| size | 尺寸 | `enum` | `md` |
| suffix | - | `ReactNode` | `-` |
| title | 标题 | `ReactNode` | `-` |
| value | 当前值 | `ReactNode` | `-` |
