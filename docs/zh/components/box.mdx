# Box 容器

## 基础用法

```tsx preview
import { Box, Text } from '@koi-ui/core';
export default () => <Box p={6} border rounded="md"><Text>容器</Text></Box>;
```

## API

### Box

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| bg | - | `enum` | `-` |
| border | - | `boolean \|​ undefined` | `-` |
| p | - | `enum` | `-` |
| rounded | - | `enum` | `-` |
