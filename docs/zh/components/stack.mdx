# Stack 堆叠

## 基础用法

```tsx preview
import { Stack, Text } from '@koi-ui/core';
export default () => (
  <Stack gap={2}>
    <Text weight="bold">标题</Text>
    <Text muted>描述</Text>
  </Stack>
);
```

## API

### Stack

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| align | - | `enum` | `-` |
| direction | 排列方向 | `enum` | `-` |
| gap | 间距 | `enum` | `-` |
| justify | - | `enum` | `-` |
| responsive | 是否响应式 | `boolean \|​ undefined` | `-` |
