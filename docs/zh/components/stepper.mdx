# Stepper 步进器

## 基础用法

```tsx preview
import { Stepper } from '@koi-ui/core';
import { useState } from 'react';
export default () => { const [v,setV]=useState(1); return <Stepper value={v} onChange={setV} />; };
```

## API

### Stepper

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| className | 类名 | `string \|​ undefined` | `-` |
| defaultValue | 默认值 | `number \|​ undefined` | `0` |
| disabled | 是否禁用 | `boolean \|​ undefined` | `false` |
| label | 标签 | `string \|​ undefined` | `-` |
| max | - | `number \|​ undefined` | `-` |
| min | - | `number \|​ undefined` | `-` |
| onChange | 变化回调 | `((value: number) => void) \|​ undefined` | `-` |
| size | 尺寸 | `enum` | `md` |
| step | - | `number \|​ undefined` | `1` |
| value | 当前值 | `number \|​ undefined` | `-` |
