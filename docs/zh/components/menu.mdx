# Menu 菜单

## 基础用法

```tsx preview
import { Menu } from '@koi-ui/core';
export default () => <Menu items={[{ key: '1', label: '导航一' }, { key: '2', label: '导航二' }]} defaultSelectedKey="1" />;
```

## API

### Menu

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| items | 数据项 | `MenuItemType[]` **(必填)** | `-` |
| defaultOpenKeys | - | `string[] \|​ undefined` | `[]` |
| defaultSelectedKey | - | `string \|​ undefined` | `-` |
| mode | - | `enum` | `vertical` |
| onOpenChange | - | `((openKeys: string[]) => void) \|​ undefined` | `-` |
| onSelect | 选择回调 | `((key: string) => void) \|​ undefined` | `-` |
| openKeys | - | `string[] \|​ undefined` | `-` |
| selectedKey | - | `string \|​ undefined` | `-` |
