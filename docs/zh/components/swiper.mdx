# Swiper 滑块视图

## 基础用法

```tsx preview
import { Swiper } from '@koi-ui/core';
export default () => (
  <Swiper>
    <div className="h-32 bg-muted flex items-center justify-center text-lg font-medium">Slide 1</div>
    <div className="h-32 bg-primary/10 flex items-center justify-center text-lg font-medium">Slide 2</div>
    <div className="h-32 bg-muted/80 flex items-center justify-center text-lg font-medium">Slide 3</div>
    <div className="h-32 bg-primary/20 flex items-center justify-center text-lg font-medium">Slide 4</div>
  </Swiper>
);
```

## 箭头切换

```tsx preview
import { Swiper } from '@koi-ui/core';
export default () => (
  <Swiper showArrows>
    <div className="h-32 bg-muted flex items-center justify-center text-lg font-medium">Slide 1</div>
    <div className="h-32 bg-primary/10 flex items-center justify-center text-lg font-medium">Slide 2</div>
    <div className="h-32 bg-muted/80 flex items-center justify-center text-lg font-medium">Slide 3</div>
    <div className="h-32 bg-primary/20 flex items-center justify-center text-lg font-medium">Slide 4</div>
  </Swiper>
);
```

## API

### Swiper

滑块视图：Pointer 跟手拖拽，可选箭头 / 指示点。
loop 时用首尾克隆做无缝循环，避免末页硬切回首页。

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| autoplay | 自动播放 | `boolean \|​ undefined` | `false` |
| interval | 自动播放间隔（毫秒） | `number \|​ undefined` | `3000` |
| loop | 是否循环 | `boolean \|​ undefined` | `true` |
| onChange | 变化回调 | `((index: number) => void) \|​ undefined` | `-` |
| showArrows | 显示箭头 | `boolean \|​ undefined` | `false` |
| showDots | 显示指示点 | `boolean \|​ undefined` | `true` |
| threshold | 滑动切换阈值 | `number \|​ undefined` | `40` |
