# Badge 徽标

## 基础用法

```tsx preview
import { Badge, Button } from '@koi-ui/core';
export default () => <Badge count={5}><Button>消息</Button></Badge>;
```

## API

### Badge

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| count | 数量 | `number \|​ undefined` | `-` |
| dot | - | `boolean \|​ undefined` | `-` |
| max | - | `number \|​ undefined` | `99` |
| showZero | - | `boolean \|​ undefined` | `false` |
| size | 尺寸 | `enum` | `-` |
| variant | 变体 | `enum` | `-` |
