# Badge 徽标

## 基础用法

```tsx preview
import { Badge, Button, Space } from '@koi-ui/core';
export default () => (
  <Space>
    <Badge count={5}><Button>消息</Button></Badge>
    <Badge count={3} variant="soft"><Button variant="soft">Soft</Button></Badge>
    <Badge count={9} variant="outline"><Button variant="outline">Outline</Button></Badge>
  </Space>
);
```

## API

### Badge

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| count | 数量 | `number \|​ undefined` | `-` |
| dot | - | `boolean \|​ undefined` | `-` |
| max | - | `number \|​ undefined` | `99` |
| showZero | - | `boolean \|​ undefined` | `false` |
| size | 尺寸 | `enum` | `-` |
| variant | 变体 | `enum` | `-` |
