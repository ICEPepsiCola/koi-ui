# Divider 分割线

## 基础用法

```tsx preview
import { Divider } from '@koi-ui/core';
export default () => <Divider>分割文字</Divider>;
```

## API

### Divider

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| dashed | - | `boolean \|​ undefined` | `false` |
| orientationMargin | - | `string \|​ number \|​ undefined` | `-` |
| plain | - | `boolean \|​ undefined` | `true` |
| type | - | `enum` | `horizontal` |
