# InfiniteScroll 无限滚动

## 基础用法

```tsx preview
import { InfiniteScroll, List } from '@koi-ui/core';
export default () => <InfiniteScroll hasMore={false} loadMore={async()=>{}}><List items={[{key:'1',title:'项目1'},{key:'2',title:'项目2'}]} /></InfiniteScroll>;
```

## API

### InfiniteScroll

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| hasMore | - | `boolean` **(必填)** | `-` |
| loadMore | - | `() => void \|​ Promise<void>` **(必填)** | `-` |
| endText | - | `ReactNode` | `没有更多了` |
| loadingText | - | `ReactNode` | `加载中...` |
| threshold | - | `number \|​ undefined` | `100` |
