
# Icon 图标

## 基础用法

```tsx preview
import { Icon, Space } from '@koi-ui/core';
export default () => (
  <Space gap={4}>
    <Icon name="search" />
    <Icon name="check" />
    <Icon name="close" />
  </Space>
);
```

## 常用图标

```tsx preview
import { Icon, Space } from '@koi-ui/core';
export default () => (
  <Space gap={6}>
    <Icon name="home" />
    <Icon name="user" />
    <Icon name="settings" />
    <Icon name="bell" />
  </Space>
);
```
