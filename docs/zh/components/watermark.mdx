# Watermark 水印

## 基础用法

```tsx preview
import { Watermark } from '@koi-ui/core';
export default () => <Watermark content="Koi UI"><div className="h-32 flex items-center justify-center border border-border rounded-md">水印区域</div></Watermark>;
```

## API

### Watermark

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| content | 内容 | `ReactNode` | `Koi UI` |
| fontColor | - | `string \|​ undefined` | `rgba(0,0,0,0.08)` |
| fontSize | - | `number \|​ undefined` | `14` |
| gap | 间距 | `[number,​ number] \|​ undefined` | `[100, 100]` |
| rotate | - | `number \|​ undefined` | `-22` |
| zIndex | - | `number \|​ undefined` | `9` |
