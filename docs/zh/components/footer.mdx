# Footer 页脚

## 基础用法

```tsx preview
import { Footer } from '@koi-ui/core';
export default () => <Footer>© 2026 Koi UI</Footer>;
```

## 链接与版权

```tsx preview
import { Footer } from '@koi-ui/core';
export default () => (
  <Footer
    links={[
      { key: 'docs', label: '文档', href: '#' },
      { key: 'github', label: 'GitHub', href: '#' },
      { key: 'changelog', label: '更新日志', href: '#' },
    ]}
    copyright="© 2026 Koi UI. All rights reserved."
  />
);
```
