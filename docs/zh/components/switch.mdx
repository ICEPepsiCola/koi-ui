# Switch 开关

## 基础用法

```tsx preview
import { Switch } from '@koi-ui/core';
import { useState } from 'react';
export default () => { const [v,setV]=useState(true); return <Switch checked={v} onChange={setV} />; };
```

## API

### Switch

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| checked | 是否选中 | `boolean \|​ undefined` | `-` |
| defaultChecked | 默认选中 | `boolean \|​ undefined` | `false` |
| onChange | 变化回调 | `((checked: boolean) => void) \|​ undefined` | `-` |
| size | 尺寸 | `enum` | `md` |
