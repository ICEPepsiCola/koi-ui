
# Popup 弹出层

## 基础用法

```tsx preview
import { Popup, Button } from '@koi-ui/core';
import { useState } from 'react';
export default () => { const [o,setO]=useState(false); return <><Button onClick={()=>setO(true)}>弹出</Button><Popup open={o} onClose={()=>setO(false)}>弹出内容</Popup></>; };
```
