# FloatButton 悬浮按钮

## 基础用法

```tsx preview
import { FloatButton, Icon } from '@koi-ui/core';
export default () => <FloatButton icon={<Icon name="plus" />} />;
```

## API

### FloatButton

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| description | 描述内容 | `ReactNode` | `-` |
| icon | 图标 | `ReactNode` | `-` |
| placement | 弹出位置 | `enum` | `-` |
| size | 尺寸 | `enum` | `-` |
| variant | 变体 | `enum` | `-` |
