
# Upload 上传

## 基础用法

```tsx preview
import { Upload } from '@koi-ui/core';
export default () => <Upload accept="image/*" />;
```

## 自定义提示

```tsx preview
import { Upload } from '@koi-ui/core';
export default () => <Upload accept="image/*" hint="支持 JPG/PNG，单文件不超过 5MB" />;
```
