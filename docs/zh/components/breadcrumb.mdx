# Breadcrumb 面包屑

## 基础用法

```tsx preview
import { Breadcrumb } from '@koi-ui/core';
export default () => <Breadcrumb items={[{ title: '首页' }, { title: '列表' }, { title: '详情' }]} />;
```

## API

### Breadcrumb

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| items | 数据项 | `BreadcrumbItem[]` **(必填)** | `-` |
| separator | - | `ReactNode` | `/` |
