# Calendar 日历

## 基础用法

```tsx preview
import { Calendar } from '@koi-ui/core';
export default () => <Calendar />;
```

## API

### Calendar

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| defaultValue | 默认值 | `Date \|​ undefined` | `-` |
| disabledDate | - | `((date: Date) => boolean) \|​ undefined` | `-` |
| onChange | 变化回调 | `((date: Date) => void) \|​ undefined` | `-` |
| value | 当前值 | `Date \|​ undefined` | `-` |
