# Image 图片

## 基础用法

```tsx preview
import { Image } from '@koi-ui/core';
export default () => <Image src="https://picsum.photos/200/120" alt="demo" width={200} height={120} />;
```

## API

### Image

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
