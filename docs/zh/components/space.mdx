# Space 间距

## 基础用法

```tsx preview
import { Space, Button } from '@koi-ui/core';
export default () => (
  <Space>
    <Button>按钮1</Button>
    <Button>按钮2</Button>
    <Button>按钮3</Button>
  </Space>
);
```

## API

### Space

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| align | - | `enum` | `-` |
| direction | 排列方向 | `enum` | `-` |
| size | 尺寸 | `enum` | `-` |
| split | - | `ReactNode` | `-` |
| wrap | 是否换行 | `boolean \|​ undefined` | `-` |
