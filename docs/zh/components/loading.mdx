# Loading 加载

## 基础用法

```tsx preview
import { Loading, Button } from '@koi-ui/core';
import { useState } from 'react';
export default () => { const [o,setO]=useState(false); return <><Button onClick={()=>setO(true)}>加载</Button><Loading open={o} /></>; };
```

## API

### Loading

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| className | 类名 | `string \|​ undefined` | `-` |
| fullscreen | - | `boolean \|​ undefined` | `true` |
| open | 是否显示 | `boolean \|​ undefined` | `true` |
| tip | - | `ReactNode` | `-` |
