
# Timeline 时间轴

## 基础用法

```tsx preview
import { Timeline } from '@koi-ui/core';
export default () => (
  <Timeline
    items={[
      { key: '1', children: '创建项目' },
      { key: '2', children: '上线发布' },
      { key: '3', children: '持续迭代' },
    ]}
  />
);
```

## 标签与颜色

```tsx preview
import { Timeline } from '@koi-ui/core';
export default () => (
  <Timeline
    items={[
      { key: '1', label: '2024-01', children: '立项启动', color: 'primary' },
      { key: '2', label: '2024-03', children: '内测通过', color: 'success' },
      { key: '3', label: '2024-06', children: '回滚修复', color: 'destructive' },
      { key: '4', label: '2024-07', children: '正式发布', color: 'default' },
    ]}
  />
);
```

## 进行中

```tsx preview
import { Timeline } from '@koi-ui/core';
export default () => (
  <Timeline
    pending="更多记录加载中..."
    items={[
      { key: '1', label: '今天', children: '提交设计稿' },
      { key: '2', label: '昨天', children: '评审通过' },
    ]}
  />
);
```
