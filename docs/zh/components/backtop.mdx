# BackTop 回到顶部

## 基础用法

```tsx preview
import { BackTop } from '@koi-ui/core';
export default () => <div className="h-40 overflow-auto relative"><div className="h-96 p-4">滚动区域</div><BackTop /></div>;
```

## API

### BackTop

| 属性 | 描述 | 类型 | 默认值 |
| :---: | :---: | :---: | :---: |
| target | - | `(() => HTMLElement \|​ Window \|​ null) \|​ undefined` | `-` |
| visibilityHeight | - | `number \|​ undefined` | `400` |
