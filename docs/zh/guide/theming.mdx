# 主题定制

Koi UI 主题基于 CSS 变量。可用内置皮肤（`data-theme`），也可在业务里覆盖 Token。

## 内置主题

`@koi-ui/tokens` 提供 6 套皮肤：`light`、`dark`、`ocean`、`forest`、`sunset`、`violet`。

```tsx preview
import { KoiProvider, Button, Alert, Tag, Space } from '@koi-ui/core';

export default () => (
  <KoiProvider theme={{ name: 'ocean' }}>
    <Space>
      <Button color="primary">主按钮</Button>
      <Button color="primary" variant="soft">轻量</Button>
      <Button color="primary" variant="outline">描边</Button>
      <Tag>标签</Tag>
      <Alert title="海洋主题" description="通过 KoiProvider 应用 data-theme" />
    </Space>
  </KoiProvider>
);
```

文档演示区在桌面/移动切换旁提供主题切换。

## Provider 覆盖

`KoiProvider.theme` 会把变量写到主题根节点：

```tsx pure
import { KoiProvider } from '@koi-ui/core';

<KoiProvider theme={{ name: 'violet', primaryColor: 'hsl(280 80% 50%)', radiusMd: '1rem' }}>
  {children}
</KoiProvider>
```

## 手动覆盖 CSS

```css
:root {
  --color-primary: hsl(160 84% 39%);
  --color-primary-foreground: hsl(0 0% 100%);
  --radius-md: 0.75rem;
}

html[data-theme='ocean'] {
  /* 或任意子树 */
}
```

## 暗色模式

暗色 Token 跟随 `html.dark` / `html.rp-dark` / `data-theme="dark"`：

```css
html.dark {
  --color-surface: hsl(222 47% 11%);
  --color-surface-foreground: hsl(210 40% 98%);
}
```

默认通过 `@koi-ui/core/styles.css` 引入；若只要 tokens：

```css
@import "@koi-ui/tokens/theme.css";
```

## 语义色

| Token | 用途 |
|-------|------|
| `--color-primary` | 品牌色 |
| `--color-success` / `--color-warning` / `--color-info` | 状态色 |
| `--color-destructive` | 危险 |
| `--color-surface` / `--color-muted` / `--color-border` | 表面 |
| `--radius-sm/md/lg` | 圆角 |

## 外观读法

跨组件统一：

- **solid / primary** — 实心
- **soft** — 浅底
- **outline** — 描边
- **ghost** — 幽灵/弱化

`Button`：`primary | soft | outline | ghost | secondary | destructive`。`Alert`：`appearance="soft" | "outline" | "solid"` + 状态 `variant`。

## 断点 Provider

`KoiProvider` 可改移动端分界断点（默认 `lg` = 1024px）：

```tsx preview
import { KoiProvider, Button } from '@koi-ui/core';

export default () => (
  <KoiProvider breakpoint="md">
    <Button>断点改为 md (768px)</Button>
  </KoiProvider>
);
```
